# 우분투에서 Intellij 설치 및 세팅하기

## 1. 설치파일 다운로드

https://www.jetbrains.com/idea/download/#section=windows

## 2. 설치하기
```
$ sudo tar xf -*.tar.gz -C /opt/
```

## 3. 실행
```
$ cd /opt/-*/bin
```

```
$ ./idea.sh
```

## 4. 플러그인 세팅

- ignore : gitignore 생성 플러그인
- lombok : lombok 플러그인
- material theme ui : 테마
